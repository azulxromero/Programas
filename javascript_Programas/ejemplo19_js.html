DISEÑO CSS DE CAJAS FLEXIBLE (FLEXBOX)

En este ejemplo se ha creado un encabezado, dos columnas/cuadros y un pie de página (footer). En pantallas más pequeñas, las columnas se apilarán encima de la otra.

Puede cambiar el tamaño de la ventana el navegador para ver el efecto responsivo.

El uso de flexbox garantiza que los elementos se comporten de manera predecible cuando el diseño de la página deba adaptarse a diferentes tamaños de pantalla y diferentes dispositivos de visualización.

Nota: Flexbox no es compatible con el Internet explorer 10 y versiones anteriores.
