Las ventajas de tener un archivo js externos son :

  • Separa HTML y el código js
  • Hace que HTML y JavaScript sean más fáciles de leer y mantener.
  • Los archivos JavaScript almacenados en caché pueden acelerar la carga de la página
  • Se pueden agregar varios archivos de script a una página, use varias etiquetas de script
