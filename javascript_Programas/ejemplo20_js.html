FORMULARIOS

Se utiliza un formulario HTML para recopilar la entrada del usuario. La entrada del usuario se envía con mayor frecuencia a un servidor para su procesamiento.

El formulario es un contenedor para diferentes tipos de elementosde entrada, como: campos de texto, casillas de veriicación, botones de opción, botones de envío, etc.

El atributo del método

El atributo method, especifica el método HTTP que se utilizará al enviar los datos del formulario.
Los datos del formulario se pueden enviar como variables de URL (con method="get") o como transacción posterior HTTP (con method="post").
El método HTTP predeterminado al enviar datos de formulario es GET.
