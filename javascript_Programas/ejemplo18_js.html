Funciones de JavaScript

Sintaxis de funciones de JavaScript

Una función de JavaScript se define como la palabra clave function, seguida de un nombre, seguido de un paréntesis (): Los nombres de funciones pueden contener letras, dígitos, subrayados y signos de dólar (las mismas reglas que las variables). Los paréntesis pueden incluir nombres de parámetros separados por comas: (parámetro1, parámetro2, . . .) El código a ejecutar, por la función, se coloca entre corchetes: {}

Los parámetros de la función se enumeran entre paréntesis () en la definición de la función. Los argumentos de la función son los valores recibidos por la función cuando se invoca. Dentro de la función, los argumentos (los parámetros) se comportan como variables locales.

Cómo se invoca a una función

El código dentro de la función se ejecutará cuando "algo" invoque (llame) a la función:

  • Cuando ocurre un evento (cuando un usario hace clic en un botón)
  • Cuando se invoca (llama) desde código JavaScript
  • Automáticamente (auto invocado)

Función Return (retorno)

Las funcions suelen calcular un valor de retorno. El valor de retorno se "devuelve" a la "persona que llama":

Ejemplo

Ejemplo: function producto(p1*p2)
return p1*p2
se hace la llamada producto (5, 9)


¿Por qué usar funciones?

Porque puede reutilizar el código: defina el código una vez y utilícelo muchas veces. Puede usar el mismo código muchas veces con diferentes argumentos para producir resultados diferentes.

En este ejemplo se llama a una función para convertir de Fahrenheit a Celsius:

function aCelcius(f){
return (5/9) * (f-32) }
document.getElementById("demo2").innerHTML = aCelcius(77);

Variables locales

Las variables declaradas dentro de una función de JavaScript se vuelven LOCALES para la función. Solo se puede acceder a las variables locales desde dentro de la función.

Dado que las variables locales solo se reconocen dentro de sus funciones, las variables con el mismo nombre se pueden usar en diferentes funciones. Las variables locales se crean cuando se inicia una función y se eliminan cuando se completa la función.
